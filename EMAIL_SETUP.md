## Инструкция по настройке

### Шаг 1: Регистрация на Web3Forms

1. Откройте https://web3forms.com/
2. Нажмите **"Get Started Free"**
3. Введите email куда будут приходить заявки: **info@lasercut.ru**
4. Подтвердите email (проверьте почту)
5. Скопируйте **Access Key** (выглядит как: `a1b2c3d4-e5f6-7890-abcd-ef1234567890`)

### Шаг 2: Добавьте Access Key в формы

Замените `YOUR_WEB3FORMS_ACCESS_KEY` на ваш ключ в файлах:

1. **pages/contacts.html** (строка 52)
2. **pages/reviews.html** (строка 20)

Найдите строку:
```html
<input type="hidden" name="access_key" value="YOUR_WEB3FORMS_ACCESS_KEY">
```

Замените на:
```html
<input type="hidden" name="access_key" value="ВАШ_КЛЮЧ_ЗДЕСЬ">
```

### Шаг 3: Закоммитьте и запушьте изменения

```bash
git add pages/contacts.html pages/reviews.html
git commit -m "Настроил формы с Web3Forms"
git push origin main
```

### Шаг 4: Проверьте работу

1. Откройте сайт после деплоя (2-3 минуты)
2. Заполните форму на странице `/contacts/`
3. Проверьте почту **info@lasercut.ru**

## Что будет приходить на почту

### Форма обратной связи:
- Тема: "Новая заявка с сайта"
- Имя клиента
- Телефон
- Email
- Сообщение
- Прикреплённый файл (если есть)

### Форма отзывов:
- Тема: "Новый отзыв с сайта"
- Имя
- Компания
- Email

- Текст отзыва

## Настройки Web3Forms (опционально)

В личном кабинете Web3Forms можно:
- Изменить email получателя
- Добавить несколько email для копии
- Настроить автоответ клиенту
- Включить уведомления в Telegram/Slack

## Лимиты бесплатного плана

- 250 заявок в месяц
- Файлы до 5 МБ
- Базовая защита от спама

Если нужно больше — есть платные планы от $5/месяц.

## Альтернативы Web3Forms

Если нужны другие варианты:
- **Formspree** (https://formspree.io/) — 50 заявок/месяц бесплатно
- **Getform** (https://getform.io/) — 100 заявок/месяц
- **EmailJS** (https://www.emailjs.com/) — отправка через JavaScript

## Проблемы и решения

**Заявки не приходят:**
- Проверьте Access Key
- Проверьте папку "Спам"
- Убедитесь что email подтверждён

**Файлы не прикрепляются:**
- Проверьте размер файла (макс 5 МБ)
- Проверьте формат (DXF, DWG, PDF, SVG)

**Форма не отправляется:**
- Откройте консоль браузера (F12)
- Проверьте ошибки JavaScript
- Проверьте заполнение обязательных полей
