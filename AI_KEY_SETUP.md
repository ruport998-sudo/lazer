# Установка AI-консультанта

## Ключ уже настроен!

API ключ Groq находится в файле `assets/js/api-key.js` и готов к работе.

**Файл защищён:**
- ✅ Не попадает в Git (`.gitignore`)
- ✅ Не индексируется поисковиками (`robots.txt`)
- ✅ AI-консультант работает напрямую с Groq API

## Как это работает:

1. Пользователь открывает чат на сайте
2. Сообщение отправляется напрямую в Groq API
3. Ответ приходит мгновенно
4. Всё работает без Cloudflare Worker

## Если нужно изменить ключ:

Отредактируйте файл `assets/js/api-key.js`:

```javascript
const API_CONFIG = {
  GROQ_API_KEY: 'ваш_новый_ключ'
};
```

## Получить новый ключ:

https://console.groq.com/keys
